# 🎮 Mini Video Game in C++

This project is a 2D survival game developed from scratch as part of my computer science studies at the **University of Paris-Saclay (IUT d'Orsay)**.

## 📝 Project Overview
The goal was to design and implement core gameplay mechanics. This project allowed me to strengthen my skills in **Object-Oriented Programming (OOP)** and logic structure.

### Key Features:
* **Movement & Logic:** Smooth player controls and game event handling.
* **Gameplay Elements:** Avoiding enemies and collecting coins to increase the score.
* **Debugging:** Full management of the development lifecycle, from conceptualization to bug fixing.

## 🛠️ Technical Stack
* **Language:** C++
* **Environment:** MSYS2 / g++ compiler
* **Tools:** Git Bash for version control

## 🚀 How to Run
To compile and run the game on your machine (ensure you have a C++ compiler installed):

1. Clone the repository:
   ```bash
   git clone [https://github.com/GloryBamazi/cpp-mini-game.git](https://github.com/GloryBamazi/cpp-mini-game.git)```

2.Compile the source code:
```Bash
g++ main.cpp -o mygame
```
3.Launch the executable:
```Bash
./mygame
